Recently ive found myself taking over a number of non-maintained documents with the goal of updating and up-issuing them.

One of the headaches ive come across is that the original author did not use the auto captions provided in word for tables etc. so inserting new tables or moving tables requires manually updating all of the other table numbers.

Or does it. . . . .

With a little regular expression & field code magic all the changes only took 5 minutes. Using regular expressions to find the unwanted text is easy enough, replacing it with the correct field codes that would continue on the numbering correctly was a little convoluted but a quick google returned this which sugested using ^c as a pointer to the clipboard and copying the required text for replacement.

Saved me more than enough time for posting an update to share this trick with everyone else
